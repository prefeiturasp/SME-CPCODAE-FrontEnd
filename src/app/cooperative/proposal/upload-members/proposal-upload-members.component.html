INSTRUÇÕES PARA IMPORTAÇÃO DOS DADOS DOS COOPERADOS PARTICIPANTES DO PROJETO DE VENDA
  1. Clique aqui para baixar o modelo de planilha
  2. Preencha os dados em cada coluna, atendendo às seguintes observações:
    • Na coluna DAP/CAF e CPF, certifique-se de que não há espaços em branco ou outros caracteres antes ou depois dos valores inseridos;
    • Os números de CPF deverão conter pontos e traço de separação;
    • Na coluna QUANTIDADE, informe apenas o valor numérico do quantitativo com o qual cada associado irá participar (sem unidade de peso ou volume – KG, L, etc);
    • A quantidade proposta para cada associado poderá conter números decimais, contudo, a quantidade total do projeto deverá ser um número inteiro.
    • Na coluna PREÇO, insira apenas o valor numérico do item (sem símbolos de R$, $ ou outros caracteres); o preço inserido deverá ser o preço estabelecido conforme o edital; para produtos orgânicos sem especificação de preço no edital, poderá ser acrescentado até 30% sobre o valor do item convencional.
    • O valor total da proposta de cada associado não deve ultrapassar o limite de R$ 40.000,00, bem como o valor total do projeto não deve ultrapassar o valor previsto na Chamada.
  3. Salve a planilha
  4. Clique em Subir a Planilha para carregá-la no Sistema. Caso haja alguma incorreção, o Sistema emitirá uma lista de itens a serem corrigidos. Efetue as correções necessárias e siga, novamente, os passos 1, 2, 3 e 4.
    - Selecionado: {{ selectedFiles[0].name }}
  5. Estando as informações corretas, clique no botão "IMPORTAR" para enviar a planilha para o Sistema.
