INSTRUÇÕES PARA IMPORTAÇÃO DOS DADOS DOS COOPERADOS PARTICIPANTES DO PROJETO DE VENDA
  1. Clique aqui para baixar o modelo de planilha
  2. Preencha os dados em cada coluna, atendendo às seguintes observações:
    • Na coluna DAP/CAF e CPF, certifique-se de que não há espaços em branco ou outros caracteres antes ou depois dos mesmos;
    • Na coluna QUANTIDADEinforme apenas o valor numérico do quantitativo com o qual cada associado irá participar (sem unidade de peso ou volume – KG, L, etc);
    • Na coluna PREÇO, insira aapenas o valor numérico do item (sem símbolos ou outros caracteres – R$), conforme especificado no Edital.
    • O valor total da proposta de cada associado não deve ultrapassar o limite de R$ 40.000,00, bem como o valor total do projeto não deve ultrapassar o valor previsto na Chamada.
  3. Salve a planilha
  4. Volte para a tela do Sistema e clique em "Subir a Planilha" para carregá-la no Sistema. Caso haja alguma incorreção, o Sistema emitirá uma lista de itens a serem corrigidos. Efetue as correções necessárias e siga, novamente, os passos 1, 2, 3 e 4.

  5. Estando as informações corretas, clique no botão "IMPORTAR" para enviar a planilha para o Sistema.
